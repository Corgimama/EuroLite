# EuroLite
It's Eurovision voting imitation. Your hosting must support PHP and MySQL and... done. File ".env" contains necessary passwords. Deal it with your friends-eurofans.

Это имитация голосования в финале Евровидения. Используемый хостинг должен поддерживать PHP и MySQL и ... в принципе всё на том. Файл ".env" содержит необходимые пароли - делитесь ими со своими друзьями еврофанами.

## Роли: администратор и зритель.

### Администратор:
- готовит голосование (кнопка "Новый финал"):
  - создаёт финалистов:
    - выбирает флаг (если флага нет, внести изменения в каталог "flags" и скрипт admin.js)
    - вводит имя исполнителя
    - вводит название песни- 
  - располагает финалистов в порядке выступление
- запускает голосование (кнопка "Начать голосование")
- завершает голосование (кнопка "Завершить голосование")
- смотрит итоговую таблицу (возможно сокрытие каких-либо зрителей и пересчёт баллов)

### Зритель:
- ожидает начала голосования
- участвует в голосовании:
  - расставляет финалистов по таблице баллов (всё как на Евровидении, но одиннадцатая строка - 0 баллов - начисляются тому, кто меньше всех впечатлил зрителя)
  - подписывает своё имя
  - отправляет свои оценки
- смотрит итоговую таблицу (показана только сумма баллов, детали скрыты)
